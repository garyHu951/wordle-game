## 🎮 功能改進

### 核心變更
**從各自單字 → 共享單字**

#### 修改前
- 玩家A猜單字A
- 玩家B猜單字B
- 無法直接比較速度

#### 修改後
- 玩家A和玩家B猜同一個單字
- 直接競爭誰更快猜對
- 更公平的競技環境

## 🛠️ 技術修改詳情

### 後端修改 (backend/server.js)

#### 1. 房間數據結構
```javascript
// 從
currentWords: {} // 每個玩家的單字

// 改為
currentWord: null // 共享單字
```

#### 2. 回合開始邏輯
- 為每個回合生成一個共享單字
- 雙方玩家使用相同單字

#### 3. 猜測驗證
- 使用共享單字進行驗證
- 回合結束後清除共享單字

#### 4. 答案顯示
- 顯示共享單字作為答案
- 雙方看到相同的答案
