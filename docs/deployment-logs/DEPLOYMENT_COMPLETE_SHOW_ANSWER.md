## 🎯 新功能說明

### 在對戰模式中：
1. **進入對戰模式** → 創建或加入房間
2. **開始遊戲** → 在遊戲頁面右上角找到 "SHOW ANSWER" 按鈕
3. **點擊按鈕** → 顯示當前回合的目標答案
4. **再次點擊** → 隱藏答案 ("HIDE ANSWER")
5. **新回合** → 答案自動重置

### 功能特色：
- 🎨 **像素風格設計** - 符合遊戲整體風格
- 🔒 **安全機制** - 只顯示自己的答案，不洩露給對手
- 🎵 **音效反饋** - 點擊按鈕有相應音效
- ♻️ **自動重置** - 新回合開始時自動隱藏
