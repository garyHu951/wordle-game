# 🎮 Wordle Game - 像素風格猜字遊戲

## 🌟 網站特色概覽

### 🎨 **獨特的像素風格設計**
- **復古像素美學**: 採用經典的像素風格界面，搭配 Press Start 2P 字體
- **動態視覺效果**: 豐富的像素風格動畫和過渡效果
- **色彩豐富**: 綠色(正確)、黃色(存在)、灰色(錯誤)的直觀顏色系統
- **響應式設計**: 完美適配桌面和移動設備

### 🎵 **沉浸式音效體驗**
- **背景音樂系統**: 
  - 主頁面專屬背景音樂
  - 單人模式專屬音樂
  - 對戰模式專屬音樂
- **豐富的音效反饋**:
  - 按鍵音效 (一般、取消、跳過)
  - 方塊顏色音效 (綠、黃、灰)
  - 遊戲事件音效 (得分、對戰開始)
- **智能音效管理**: 防重複播放、音量控制、靜音功能

## 🎯 遊戲模式

### 🧩 **單人模式 (Single Player)**
- **多種單字長度**: 4、5、6、7 字母可選
- **固定挑戰**: 6 次猜測機會
- **即時反饋**: 每次猜測後立即顯示結果
- **答案顯示**: 隨時可查看當前答案
- **暫停功能**: 支援遊戲暫停和恢復
- **字母狀態追蹤**: 實時顯示已猜測字母的狀態

### ⚔️ **競賽模式 (Competitive)**
- **即時對戰**: 兩名玩家同時進行遊戲
- **獨立回合系統**: 每位玩家有自己的進度，但共享相同單字
- **房間系統**: 創建或加入遊戲房間
- **實時同步**: 雙方遊戲狀態即時更新
- **暫停協調**: 任一玩家可暫停遊戲
- **跳過回合**: 可選擇跳過困難回合並查看答案
