## 🧪 功能測試指南

### 單人模式測試
1. **暫停功能**:
   - 按P鍵或點擊暫停按鈕
   - 確認遊戲暫停，顯示覆蓋層
   - 按P鍵或點擊恢復按鈕
   - 確認3秒倒計時後恢復

2. **鍵盤響應**:
   - 暫停期間無法輸入
   - 恢復後立即響應鍵盤

### 競賽模式測試
1. **鍵盤輸入修復**:
   - 答對一個回合後
   - 立即測試鍵盤輸入
   - 確認無延遲響應

2. **跳過回合答案**:
   - 點擊跳過回合按鈕
   - 確認顯示當前答案2秒
   - 自動進入下一回合

3. **暫停功能**:
   - 一個玩家點擊暫停
   - 確認顯示等待對手同意
   - 對手也點擊暫停
   - 確認遊戲暫停
   - 雙方點擊恢復
   - 確認3秒倒計時後恢復

4. **錯誤信息**:
   - 輸入不存在的單字
   - 確認顯示英文錯誤信息
