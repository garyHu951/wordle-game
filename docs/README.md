# 文檔目錄

這個目錄包含了項目開發過程中的各種文檔和工具。

## 目錄結構

### 📁 deployment-logs/
包含所有部署相關的日誌和狀態文檔：
- 部署狀態報告
- 功能更新記錄
- 修復日誌
- 優化記錄
- 部署指南

### 📁 development-tools/
包含開發和調試工具：
- 部署檢查腳本
- 診斷工具
- 問題解決方案
- 開發輔助腳本

## 文檔說明

這些文檔主要用於：
1. **開發記錄**: 記錄開發過程中的各種更改和決策
2. **部署追蹤**: 追蹤部署狀態和問題解決過程
3. **調試工具**: 提供各種檢查和診斷工具
4. **歷史參考**: 保存項目演進的完整記錄

## 使用建議

- **開發者**: 可以參考這些文檔了解項目的演進歷史
- **維護者**: 使用development-tools中的腳本進行系統檢查
- **用戶**: 通常不需要關注這些技術文檔

## 注意事項

這些文檔是開發過程的副產品，主要用於內部參考。如果你是普通用戶，請參考根目錄的README.md文件。
