# 答案顯示按鍵圖標更新

## 更新概述
根據用戶需求，更新了單人模式中答案顯示按鍵的圖標，使用不同的眼睛圖標來更好地區分顯示和隱藏狀態。

## 🔄 修改內容

### 修改前
```jsx
👁️ {showAnswer ? 'HIDE ANSWER' : 'SHOW ANSWER'}
```

### 修改後
```jsx
{showAnswer ? '🙈 HIDE ANSWER' : '👁️ SHOW ANSWER'}
```

## 📱 視覺效果

### 顯示答案狀態
- **圖標**: 👁️ (睜眼圖標)
- **文字**: SHOW ANSWER
- **顏色**: 藍色按鍵 (`bg-blue-600 hover:bg-blue-500`)
- **含義**: 點擊可以顯示當前題目的答案

### 隱藏答案狀態
- **圖標**: 🙈 (閉眼猴子圖標)
- **文字**: HIDE ANSWER
- **顏色**: 紅色按鍵 (`bg-red-600 hover:bg-red-500`)
- **含義**: 點擊可以隱藏當前顯示的答案

## 🎯 改進優勢

### 1. 視覺直觀性
- **睜眼圖標 (👁️)**: 直觀表示"看見"、"顯示"的概念
- **閉眼圖標 (🙈)**: 直觀表示"不看"、"隱藏"的概念
- 圖標與功能語義完美匹配

### 2. 用戶體驗提升
- **快速識別**: 用戶可以通過圖標快速識別當前狀態
- **減少認知負擔**: 不需要閱讀文字就能理解功能
- **國際化友好**: 圖標是通用語言，不受語言限制

### 3. 界面一致性
- **保持原有顏色方案**: 藍色顯示，紅色隱藏
- **保持按鍵樣式**: Pixel風格邊框和陰影效果
- **保持功能邏輯**: 點擊行為完全不變

## 🎮 用戶指南

### 如何使用
1. **顯示答案**: 點擊帶有 👁️ 圖標的藍色按鍵
2. **隱藏答案**: 點擊帶有 🙈 圖標的紅色按鍵
3. **狀態識別**: 通過圖標和顏色快速識別當前狀態

### 使用場景
- **學習模式**: 先嘗試猜測，再顯示答案檢查
- **困難時刻**: 卡關時查看答案獲得提示
- **策略研究**: 分析答案規律，改進猜測策略
