# 暫停按鍵文字簡化更新

## 更新概述
根據用戶需求，簡化了單人模式和雙人模式的暫停按鍵文字顯示。

## 實現的改進

### 🔄 暫停按鍵文字簡化
**需求**: 單人模式和雙人模式的暫停按鍵只需顯示"PAUSE"

**修改前**:
- 單人模式: 根據狀態顯示 `⏸️ PAUSE` 或 `▶️ RESUME`
- 雙人模式: 根據狀態顯示 `PAUSE` 或 `RESUME`

**修改後**:
- 單人模式: 固定顯示 `⏸️ PAUSE`
- 雙人模式: 固定顯示 `PAUSE`

## 技術實現

### 修改的代碼位置

#### 單人模式 (約第1670行)
```jsx
// 修改前
{isPaused ? '▶️ RESUME' : '⏸️ PAUSE'}

// 修改後
⏸️ PAUSE
```

#### 雙人模式 (約第2672行)
```jsx
// 修改前
{isPaused ? 'RESUME' : 'PAUSE'}

// 修改後
PAUSE
```

## 功能保持
- ✅ 按鍵的點擊功能完全保持不變
- ✅ 暫停/恢復邏輯完全保持不變
- ✅ 只是視覺顯示文字的簡化
- ✅ 按鍵樣式和顏色保持統一的黃色

## 用戶體驗改進
1. **一致性**: 按鍵文字不再變化，提供更一致的用戶界面
2. **簡潔性**: 用戶不需要理解按鍵文字的動態變化
3. **直觀性**: "PAUSE" 文字直接表達按鍵的主要功能
